HTML

Etiquetas/tags html:

  • Formato del Texto:

    • <em> La cuenta porfis (no importa el navegador, le da algún tipo de énfasis)</em>

    • <i> La cuenta porfis </i>

    • <strong> La cuenta porfis <b> </strong>

    • <b> La cuenta porfis </b>

    • <small> La cuenta porfis </small>

    • <del> La cuenta porfis </del>

    • <ins> La cuenta porfis </ins>

    • <u> La cuenta porfis </u>

    • <mark> La cuenta porfis </mark>

  • Estructura del Texto:

    • <br> Salto de linea

    • <p> Parrafo: Cada nuevo párrafo va en una linea distinta

      • Atributos

        • align: <p align=’center’ | ‘right’ | ‘left’ | ‘justify’

    • <pre> Muestra el texto literalmente como lo puse (con saltos y tabulaciones

    • <!-- Texto comentado →

    • <div> Es un elemento html genérico de bloque (y, por ej, asignarles attr.); agrega un salto de linea al cerrar

    • <span> Es un elemento html genérico de línea. No agrega salto de linea

    • <ul> / <ol> Listas / lista numeradas

      • <li> Items de cualquiera de estas listas

  • Imágenes:

    • <img>

      • Atributos

        • src: <img src=”path_to_file.jpg”

        • width=’450’ | ‘63%’

        • alt: Comentario de la imagen altr=’Papapap ddfdf’

        • hsspace: Margen de la imagen hsspace=’20’

  • Enlaces:

    • <a>

      • Atributos

        • href: <a href=”path_to_una_pagina.html”> lififgfijijdls </a>

          • <a href=”url”> lififgfijijdls </a>

        • target: target=”_blank” || “_self” | “_parent” | “_top”


CSS

Definiendo CSS mediante atributos en html

  • style: <p style=”color:green; background:yellow;”>La cuenta porfis</p>

Estilos por etiqueta

  • Esto se refiere a definir estilos para los distintos tags en el head

  • <style>

    • h1{ font-family:’Courier-new’; color:’red’; …..}

    • p{ background:’green’; align:’right’;….} </style>

Hoja de estilos

  • Se referencia en el html mediante <link rel=’stylesheet’ href=’path_to_file.css’>

Estilos por clase

  • <p class=’mi-clase’>

  • <style> .miclase { font-family:’Arial-black’; text-align:’center’ }

Selectores en CSS:

  • De tipo, de clase, de id

    • a{ estilo }, p{ estilo }, div{ estilo }

    • .mi-clase{ estilo }, .box{ estilo }

    • #un_boton{ estilo }, #una_caja{ estilo }

  • De atributo

    • a [href*=’.ar’] { estilo }

    • p [class*=’box’] { estilo }

  • De pseudo-clase y pseudo-elemento

    • p:hover { estilo }

    • p::first-line { estilo }

  • De jerarquía

    • p ol { font-weight: bolder } # Pone en negrita todos los elementos por debajo de ol

    • p > ol { font-weight: bolder } # Pone en negrita solo los hijos directos ol
